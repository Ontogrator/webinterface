Welcome To The Ontogrator

This is demonstration of how ontologies can be used to automatically generate user interfaces for browsing data sets.

Searching is performed using sets of hierarchically structured terms, i.e. ontologies.

Behind the scenes there is a database which knows about the ontologies. It has been generated by feeding text from the source data into the Terminizer and NCBO Annotator services which scan text looking for known ontogical terms.

This demonstration relies on the marvellous work done by the creators of:

  • The Terminizer
  • The NCBO Annotator
  • MySQL, PHP and Ext JS.

Developed by David Hancock for the NERC Environment Bioinformatics Centre.

Developed for use with Clinical Trials by Peter Rodda as part of a collaboration between The University of Manchester and AstraZeneca.
